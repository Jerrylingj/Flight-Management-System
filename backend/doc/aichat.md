## aichat
- /api/aichat
传入
```ts
{
    "messages":{
        "role":"assistant"|"user"|"system",
        "content":string
    }[]
}
```
成功返回
```ts
{
    "code":200,
    "message":"操作成功",
    "data":{
        "role":"assistant",
        "content":string
    }
}
```
失败返回
失败返回
```ts
{
    "code":500,
    "message":"失败原因",
    "data":null
}
```
